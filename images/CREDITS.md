# 照片授權清單

本目錄下所有從 Wikimedia Commons 下載的照片，作者與授權標示。
UI 上也可透過每張圖右下角的 (i) 按鈕看到相同資訊。

**通則**：這些照片皆可自由重用（含商用、改編），但必須保留作者名與授權標示。
